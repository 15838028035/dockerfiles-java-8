### Automation

The __generate.sh__ script may be used to generate folders and files (8u11-* / sh -x scripts/generate.sh). The __automate.sh__ script may be used to trigger image generation on the docker hub. The __validate.sh__ script may be used to validate OS and JAVA versions. The __tabulate.sh__ script may be used to generate the above table with image sizes.
